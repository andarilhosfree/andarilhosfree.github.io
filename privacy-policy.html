Política de Privacidade – Andarilhos das Terras Livres

O Andarilhos das Terras Livres acredita que a privacidade e a transparência são fundamentais. Por isso, disponibilizamos este documento para esclarecer como as informações são coletadas, utilizadas e disponibilizadas aos usuários.

Objetivo desta política

Esta política tem como objetivo informar de forma clara e transparente como os dados fornecidos pelos usuários são tratados dentro do site.

Ao fornecer seus dados, o usuário declara estar ciente e de acordo com as condições aqui descritas.

Quais dados são coletados

Os únicos dados pessoais que podem ser informados pelos usuários são:

  • Nome
  • Localização
  • Telefone

O fornecimento dessas informações é totalmente opcional.

Forma de coleta e armazenamento

Os dados são inseridos voluntariamente pelos usuários em uma planilha aberta, acessível publicamente.

Essa planilha é:

  • Gerenciada por serviços do Google
  • De acesso livre para qualquer pessoa com o link
  • Não possui controle restrito por parte do site

O Andarilhos das Terras Livres não realiza armazenamento próprio desses dados fora da planilha.

Uso e compartilhamento dos dados

Os dados informados:

  • São visíveis publicamente na planilha
  • Podem ser acessados por qualquer pessoa
  • Não são controlados, moderados ou restringidos pelo site

Dessa forma, o usuário está ciente de que, ao inserir seus dados:

  • Está voluntariamente tornando-os públicos
  • Assume os riscos relacionados à exposição dessas informações

Exclusão e controle dos dados (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (LGPD), o usuário pode, a qualquer momento:

  • Excluir seus próprios dados diretamente da planilha
  • Editar ou corrigir as informações inseridas
  • Optar por não fornecer ou remover totalmente seus dados

A gestão, edição e exclusão dos dados são de responsabilidade do próprio usuário que realizou o cadastro.

Responsabilidade sobre os dados

O Andarilhos das Terras Livres:

Não se responsabiliza pelo armazenamento, uso ou divulgação dos dados pessoais, uma vez que:

  • A planilha é pública
  • O acesso é irrestrito
  • O preenchimento é opcional

A responsabilidade pelo fornecimento, manutenção e remoção das informações é exclusivamente do usuário.

Links externos

O site pode conter links para serviços de terceiros, incluindo:

  • Plataforma do Google (planilha)
  • Grupo externo no WhatsApp

Ao acessar esses links, o usuário estará sujeito às políticas de privacidade próprias dessas plataformas.

O Andarilhos das Terras Livres não se responsabiliza pelas práticas, coleta ou uso de dados realizados por terceiros, incluindo o WhatsApp ou qualquer outro serviço externo.

Alterações nesta política

Esta Política de Privacidade pode ser atualizada a qualquer momento para refletir mudanças no funcionamento do site ou em exigências legais.

Recomendamos que o usuário revise este documento periodicamente.
